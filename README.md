# 🛒 E-Commerce Frontend Project

## 📌 Project Overview

This is a **frontend-based e-commerce web application** built using **HTML, CSS, and JavaScript**.
It allows users to browse products, view details, add items to cart, and manage wishlist.

---

## 🚀 Features

* 🏠 Home Page with product listing
* 📦 Product Details Page
* 🛒 Add to Cart functionality
* ❤️ Wishlist feature
* 💳 Checkout Page
* 📊 Dynamic data using JSON
* ⭐ Product ratings display
* 📱 Responsive design (Mobile, Tablet, Desktop)

---
